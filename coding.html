COURSES

  • CODING NURTURE

    Coding, or computer programming, is a creative process programmers perform to tell a computer how to perform a task. At its foundation, it involves writing computer programs using programming languages. Coding for kids is usually taught using content that is high-interest while creating projects that involve creative input.

    Results : By the end of the course, each student will make his/her own a simple unique game, that can be played online and shared with friends and family.

  • CODING PRODIGY

    The main objective of this course will be to learn the concepts of coding at an early stage. The process of problem solving through computer programming will be taught. Designing different algorithms and building computer programs will be taught step by step using Google’s graphics-based coding tool Blockly. The process of writing computer programs will be simplified to a step-by-step block building activity. Writing basic codes in python will be taught.

    Results : By the end of this course students will be able to make their own algorithms for solving programming problems and execute them in Blockly and Python. This will make a strong base for a career in computer science and engineering.

  • CODING GENIUS

    This course will target learning the concepts of computer programming through teaching some of the milestone algorithms in the world of computer science like binary search, quick sort, bubble sort, shortest path and their implementation using Python, the hottest programming language of the day. Starting with very basic exercises like making their own calculator using Python, the course will help students to build towards writing more advanced algorithms. Familiarizing kids with these concepts at an early stage will give them an accurate idea of what they will be studying if they choose computer science and will serve well in discovering their interest early in computer science.

    Results : By the end of this course kids will be able to use Python to implement some of the most important algorithms in computer science. This will also serve as a window for kids to look into the world of computer science to pique their interest.
